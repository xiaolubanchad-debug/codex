# Red Loom Editorial Blog

A `Next.js App Router` MVP for an editorial blog inspired by classic magazine-style homepages and rebuilt for a more modern reading experience.

## Pages

- `/` homepage
- `/posts` archive page
- `/posts/[slug]` article detail page
- `/search` search page
- `/about` about page

## Docs

- [MVP overview](./docs/mvp-overview.md)
- [UI direction](./docs/ui-direction.md)

## Local Run

```bash
npm install
npm run dev
```

## Current Scope

- local mock content data
- shared editorial card system
- client-side archive search
- responsive layouts for desktop, tablet, and mobile
