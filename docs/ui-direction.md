# UI Direction

## Reference Translation

The goal is not to copy the reference literally. The goal is to keep its editorial energy:

- clean top navigation
- strong brand presence
- a large image-like hero with an overlay card
- three-column content rhythm
- red accent lines and buttons
- a high-contrast subscription band and footer

## Visual System

### Palette

- accent red: `#d92c1f`
- dark base: `#121212`
- paper background: `#fbf8f4`
- warm neutral: `#efe7dc`
- main text: `#171717`
- secondary text: `#6a625b`

### Typography

- serif or near-serif for titles
- clean sans-serif for body copy
- tighter uppercase treatment for labels and metadata

### Mood

- editorial, not dashboard-like
- refined but not luxurious
- structured, warm, and readable

## Page Directions

### Homepage

- large hero background with a floating featured card
- one featured cluster below the hero
- section-based grids inspired by magazine layouts
- newsletter block before the footer

### Archive page

- calm header with archive purpose
- visible section filters
- card grid optimized for scanning

### Detail page

- bold article hero
- readable text column
- supporting sidebar for tags and navigation
- related reading block below the article

### Search page

- prominent search input
- suggestion tags
- dense but readable result cards
- clear empty state

### About page

- mission statement
- editorial principles
- section explanation
- collaboration/contact block

## Interaction Principles

- subtle hover motion
- large click targets for cards
- immediate search feedback
- mobile layouts should preserve hierarchy before density

## Implementation Notes

- use shared card components
- rely on CSS variables for color, spacing, and shadows
- keep feature cards visually distinct from regular cards
